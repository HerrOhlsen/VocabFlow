# Screenshots

Bilder, die im Haupt-`README.md` eingebunden werden.

## Richtlinien

- **Keine echten Schülerdaten**: Klarnamen, Moodle-IDs, E-Mails, Profilbilder unkenntlich machen oder Testkonto verwenden.
- **Format**: PNG bevorzugt, JPEG ok. SVG für UI-Diagramme.
- **Größe**: einzelne Datei < 1 MB. Bei Bedarf über [squoosh.app](https://squoosh.app) komprimieren.
- **Benennung**: kleingeschrieben, mit Bindestrichen, sprechend. Z.B. `lernansicht.png`, `editor-karten-liste.png`, `dashboard-widget.png`.
